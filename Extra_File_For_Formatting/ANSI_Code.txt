## üîπ BASIC FORMAT SYNTAX

```python
\033[<code>mYour Text\033[0m
```

* `\033[` ‚Üí start formatting
* `<code>` ‚Üí style number
* `m` ‚Üí apply
* `\033[0m` ‚Üí reset

# ‚úÖ Examples in Python

### Bold

```python
print("\033[1mBold Text\033[0m")
```

### Underline

```python
print("\033[4mUnderlined Text\033[0m")
```

### Red Text

```python
print("\033[31mRed Text\033[0m")
```

### Yellow Background + Blue Text

```python
print("\033[44;33mStylish Text\033[0m")
```

### Bold + Underline + Green

```python
print("\033[1;4;32mImportant\033[0m")
```

# üß™ Cheat Sheet Combination Format

```python
\033[STYLE;TEXTCOLOR;BACKGROUNDmText\033[0m
```

Example:

```python
print("\033[1;31;47mAlert Message\033[0m")
```

Bold Red text on White background
